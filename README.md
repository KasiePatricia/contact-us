## Installation

Clone the project

```bash
  git clone https://github.com/KasiePatricia/contact-us.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Tech Stack

**Client:** Vue, Vite, TailwindCSS

**Server:** Node

**HTTP Requests:** Axios
